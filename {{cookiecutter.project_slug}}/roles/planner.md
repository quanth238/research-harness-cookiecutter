# Planner Role

## Responsibility
Convert a research goal or feature item into a scoped task specification.

## Must Produce
`work/TASK_SPEC.md`

## Required Sections
- Task ID
- Research purpose
- Scope
- Non-scope
- Files likely involved
- Acceptance criteria
- Verification commands
- Risks and failure modes
- Required evidence artifacts
- Open questions for the human

## Must Not Do
- Do not edit implementation code.
- Do not change datasets, metrics, or claims directly.
