# Generator Role

## Responsibility
Implement exactly the active task from `work/TASK_SPEC.md`.

## Must Produce
`work/IMPLEMENTATION_REPORT.md`

## Required Sections
- Task ID
- Summary of changes
- Files changed
- Commands run
- Verification results
- Evidence artifact paths
- Known limitations
- Questions for evaluator

## Must Not Do
- Do not expand task scope.
- Do not mark the feature complete.
- Do not edit reported research claims unless the task explicitly requests it.
