# Paper And Claims Protocol

## Claim Rule
Do not add or update a scientific claim unless it points to evidence.

Allowed evidence:
- Reproducible command
- Log file
- Metric table
- Figure generation script
- Dataset checksum
- Citation
- Human-reviewed derivation

## Figures And Tables
Each figure/table should have:
- Source script
- Input artifact paths
- Output path
- Caption draft
- Verification command

## Verification Commands
- `make verify-figure`
